# Reports Previous Payslips Individual

### Report on employees payslip(s).

Reprint an employees payslip, nb uses Detailed Layout.  

**Get Employee**  
Select the employee.  

**Include**  
Enter 'Y' for the payslips you want.

**Continue**  
This will take you to the display of the payslips.
