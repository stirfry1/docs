# Employee Overview

### Summary of the employee's information

This is where the employee information is held.  
You can get detailed information from each tab.  

**Show**  
This lets you restrict the employyes to view.

**New Employee**  
Allows you to add a new employee.

**D.P.S.** Data Provisioning Service  
You can view a RTI notice FROM HMRC ie P6, SLD.  
I suggest you run this prior to each payrun.  

**,&lt;**  
Previous employee.

**.>**  
Next Employee.

**Search**  
Search for an employee.  

**Quit**  
Saves and leaves.
