# Principal Information

## How to set a Principal.
The Principal details can now be automatically entered on to each XML sent online. 

In the 'EmployerDetails/TaxOffice' tab there is a tick box to 'Send Emp Info'.  

The Surname is required if this is used.

 See also [Employer Tax Office](../employer/employerdetails/empr_taxoffice.md)
